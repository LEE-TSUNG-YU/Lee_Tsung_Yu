初始函式，input_feature_dim代表輸入訊號的維度，
hidden_feature_dim代表隱藏層的維度，設定越多代表能記住的特徵越多，不過同時也代表訓練更耗時，而且設定過多也不保證記住的都是有用的
hidden_layer_num,隱藏層的數目，層數和hidden_feature_dim相乘就是總共的特徵數

batch_size，單次輸入的批量，在pytorch裏頭，即使每次都只餵1個數據，但是在tensor的設定上還是得設定batch_size=1

定義t=0的時候的Hidden_cell_state
在LSTM裏頭，hidden state和cell state，hidden state儲存當前LSTM CELL的運算結果，並且pass到現在這一個CELL，而cell state則是儲存當前memory的值，這個值會傳遞到下一個CELL，上一個cell state會根據forget gate的值決定會不會被保留起來，如果 forget gate是0，則當前cell state就不會存有上個cell state的值。

class LSTM(nn.Module):
#建立LSTM class
    def __init__(self,input_feature_dim,hidden_feature_dim,hidden_layer_num,batch_size,classes_num):
        super(LSTM_signal,self).__init__()
        self.input_feature_dim = input_feature_dim # 輸入訊號的維度
        self.hidden_feature_dim = hidden_feature_dim # 隱藏層的維度
        self.hidden_layer_num = hidden_layer_num # 隱藏層的數目
        self.batch_size = batch_size # 單次輸入的批量
				
	#初始化LSTM       
	self.lstm = nn.LSTM(input_feature_dim,hidden_feature_dim,hidden_layer_num)
	#LSTM的輸出藉由單層的線性神經網路層分類~
	self.linear1 = nn.Linear(hidden_feature_dim,classes_num)

    def init_hidden(self):
        h0 = t.randn(self.hidden_layer_num,self.batch_size,self.hidden_feature_dim)
        c0 = t.randn(self.hidden_layer_num,self.batch_size,self.hidden_feature_dim)

'''
定義正向傳播
由於我這邊沒有要處理seq2seq的問題，因此只需要取output的最後一組元素，也就是hn
lstm的輸出會有 output和最後一個CELL的 hidden state, cell state，在pytorch裏頭，output的值其實就是每個cell的hidden state集合起來的陣列
''' 
    def forward(self,input):
        output,(hn,cn) = self.lstm(input,(h0,c0))  
        output = self.linear1(output[-1])         
        return output,(hn,cn)

# 建構LSTM物件
lstm = LSTM(input_feature_dim,hidden_feature_dim,hidden_layer_num,batch_size,5)
# 宣告反向傳播機制
optimizer = optim.Adam(lstm.parameters(),lr=0.1)
# 宣告損失函數
loss_fnc = nn.CrossEntropyLoss()


correct=0
epochs=20
for ii in range(epochs):#每一輪
    for input, label in training_data: #數據集內的每個數據和標籤
        lstm.zero_grad() #清除lstm的上個數據的偏微分暫存值，否則會一直累加
        output,(hn,cn) = lstm(input) #正向傳播    
        loss = loss_fnc(output,label) #計算loss
        loss.backward() #從loss計算反向傳播
        optimizer.step() #更新所有權種和偏差
         _, preds = t.max(output, 1) #從output中取最大的出來作為預測值
         if preds==label: #如果預測值和標籤一致
             correct+=1
