
# 股票分析網站範本

這是一個可直接部署到 GitHub Pages 的靜態股票網站。

## 功能

- 股票代碼查詢
- Plotly 股票走勢圖
- 響應式手機版 UI
- 不需要後端
- 可直接部署 GitHub Pages

---

## 使用方式

### 1. 解壓縮

### 2. 上傳到 GitHub repository

### 3. 開啟 GitHub Pages

GitHub：

Settings → Pages

Source：

Deploy from branch

Branch：

main / root

---

## 網址格式

部署後：

https://你的帳號.github.io/repository名稱/

---

## 可查詢股票

美股範例：

- AAPL
- TSLA
- NVDA
- MSFT
